## Sneha Poojary – Portfolio

##  Overview

Welcome to my personal portfolio! This is a **professional showcase of my web development skills, projects, achievements, and technical expertise**. The portfolio is built using **React.js** for a modern, responsive frontend and is **deployed on Vercel** for fast, reliable performance.

It’s designed to provide a clear, elegant view of my work, emphasizing readability, structure, and a user-friendly experience — perfect for recruiters and technical evaluators.

---

##  Technology Stack

* **Frontend:** React.js, Vite, JavaScript, HTML5, CSS3
* **Styling & Layout:** Tailwind CSS, Flexbox, CSS Grid
* **Version Control:** Git, GitHub
* **Deployment:** Vercel
* **Linting & Code Quality:** ESLint, `@eslint/js`, `eslint-plugin-react-hooks`, `eslint-plugin-react-refresh`


---

## 🎨 Features

* **About Me:** Clean, modern introduction with profile image and professional summary
* **Skills:** Comprehensive list of technical skills including web development, backend, and system design
* **Projects:** Showcase of full-stack and backend projects with descriptions, technologies used, and GitHub links
* **Certifications:** List of professional certifications with direct links
* **Achievements:** Hackathons, open-source contributions, and other notable accomplishments
* **Contact:** Easy way to reach me via email or social links

---
## ⚡ Design & UX Principles

* **Responsive Design:** Optimized for desktop, tablet, and mobile screens
* **Minimal & Elegant UI:** Focus on readability and professional presentation
* **Consistent Layout:** Uniform spacing and typography across all sections
* **Accessibility Considerations:** Semantic HTML, descriptive alt tags, and keyboard navigation

---

## 🛠 Installation & Local Setup

To run the portfolio locally:

```bash
# Clone the repository
git clone https://github.com/SnehaPoojary20/portfolio.git

# Navigate into the project
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

## 🌍 Deployment

* Deployed on **Vercel** for fast and reliable hosting
* Continuous deployment from GitHub ensures your portfolio is always up-to-date
